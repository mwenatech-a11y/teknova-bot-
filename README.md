## Jinsi ya Kuanzisha (Setup)

### 1. Sakinisha Node.js
Hakikisha una Node.js v18 au juu zaidi.

### 2. Sakinisha dependencies
```bash
cd teknova-bot
npm install
```

### 3. Anzisha bot
```bash
npm start
```

### 4. Pair namba yako
- Bot itakuuliza uweke namba yako (mfano: 255772991908)
- Utapata PAIRING CODE
- Nenda WhatsApp > Linked Devices > Link a Device
- Weka code uliyopata

### 5. Tayari!
Bot itaanza kufanya kazi. Tuma `.menu` kuona commands zote.

## Commands za Msingi
- `.menu` - Menu kuu (commands zote)
- `.help` - Msaada
- `.ping` - Check speed
- `.ai [swali]` - AI chat
- `.play [jina la wimbo]` - YouTube
- `.sticker` - Tengeneza sticker (reply picha)
- `.tagall` - Tag wote (group)

## Kuhost kwenye RENDER / RAILWAY (Cloud - hakuna keyboard)
Kwenye cloud hosting hakuna sehemu ya kuandika namba, kwa hiyo:

1. Fungua `config.js` na hakikisha namba yako iko kwenye `pairNumber` (tayari imewekwa 255772991908).
   AU weka Environment Variable jina lake `PAIR_NUMBER` na thamani namba yako (mfano 255772991908).
2. Start Command iwe: `node index.js`
3. Bot itaanza yenyewe na ku-generate PAIRING CODE kwenye **Logs**.
4. Angalia kwenye Logs, utaona:
   `PAIRING CODE: XXXX-XXXX`
5. Nenda WhatsApp > Linked Devices > Link a Device > "Link with phone number instead" > weka code.

> MUHIMU: Code ina muda mfupi (dakika chache). Ukikosa, restart deploy upate code mpya kwenye logs.
